# 更新日志

## [1.0.1] - 2026-03-14

### 改进
- ✨ 优化主脚本：运行后直接显示菜单，用户选择后自动进入对应安装
- ✨ 添加帮助信息显示功能
- 🐛 修复：删除 main 分支，统一使用 master 分支

## [1.0.0] - 2026-03-14

### 新增功能
- ✨ 主脚本：支持 curl 下载，提供交互式菜单
- ✨ 包管理器部署：使用 PyPI + uv 快速安装
- ✨ 手动部署：支持从 Release 包安装，国内镜像加速
- ✨ Docker 部署：支持 Docker Compose，国内环境优化
- ✨ 一键安装：通过 curl 直接下载并运行

### 技术特性
- 🔧 国内环境优化（镜像源加速）
- 🔧 sudo 权限智能提示
- 🔧 完整的日志和状态管理
- 🔧 支持后台运行和进程管理
- 🔧 自动依赖检查和安装
- 🔧 配置文件自动生成
- 🔧 进程管理（启动、停止、重启）

### 系统支持
- 🖥️ Linux (Debian, Ubuntu, CentOS, Arch)
- 🍎 macOS
- 💻 Windows

### 部署方式
- 📦 包管理器部署
- 📦 手动部署（Release 包）
- 🐳 Docker 部署
- 🔗 一键安装（curl）

### 文档

- 📚 config.example.yaml 配置示例
- 📚 常见问题解答
